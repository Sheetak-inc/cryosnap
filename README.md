# CryoSnap

CryoSnap is a Sheetak thermoelectric validation platform for moving from bench testing to product integration faster. It is intended to help builders, engineers, researchers, and product teams start from a working TEC control system, collect useful thermal data earlier, and reuse validated architecture where appropriate.

> **Status:** Public reference design / community edition.
>
> CryoSnap is made available to encourage personal, educational, research, evaluation, and internal use. It is **source-available**, not an OSI-approved open source project. Commercial sale, paid distribution, or integration into a commercial product requires prior written authorization from Sheetak.

## What is included

This repository may include some or all of the following:

- firmware source code
- PCB design files and schematics
- bill of materials and assembly notes
- documentation and examples
- setup, calibration, and data-collection workflows
